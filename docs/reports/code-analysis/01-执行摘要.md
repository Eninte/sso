## 项目核心功能

1. ✅ **用户认证**: 用户注册、登录、登出、密码重置
2. ✅ **JWT 令牌管理**: Access Token 和 Refresh Token 生成、验证、撤销
3. ✅ **OAuth 2.0**: 授权码流程支持、PKCE 增强安全
4. ✅ **OpenID Connect**: 发现端点、用户信息端点
5. ✅ **多因素认证**: TOTP 支持
6. ✅ **第三方登录**: Google、GitHub 社交登录
7. ✅ **管理功能**: 用户列表、禁用/启用、审计日志
8. ✅ **缓存层**: Redis 缓存支持
9. ✅ **监控指标**: Prometheus 格式指标导出
