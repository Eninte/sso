feat: 添加 SSO 服务初始化与配置向导系统

实现完整的服务初始化流程，包含配置向导和初始化面板两个阶段。

## 主要功能

### 配置向导（Setup Wizard）
- 当配置缺失时自动启动轻量级 HTTP 服务
- 提供 Web 界面配置数据库、Redis、JWT、SMTP 等
- 支持连接测试和自动生成 RSA 密钥对（3072 位）
- 保存配置到 .env 文件并自动重启服务

### 初始化面板（Init Panel）
- 服务正常运行后提供系统初始化界面
- 显示系统状态（DB、Redis、迁移状态）
- 创建首个管理员账户
- 创建默认 OAuth 客户端

## 安全改进

- 配置向导使用一次性令牌保护（32 字节随机）
- 限流保护（10 请求/分钟）
- .env 文件值转义防止注入攻击
- 文件权限设置为 0640
- 错误消息过滤，不泄露敏感信息
- 路径验证包含符号链接检测
- 密钥路径白名单支持环境变量配置（KEY_PATH_WHITELIST）
- 依赖数据库 UNIQUE 约束防止竞态条件

## 架构设计

- 完全符合 Handler → Service → Store 分层架构
- 新增 InitService 处理业务逻辑
- 使用项目工具模块（handlerutil、serviceutil、auditutil）
- 所有错误使用 apperrors 预定义错误
